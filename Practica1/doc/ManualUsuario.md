# Manual de Usuario

En este manual se desribira elfuncionamiento de la pagina html con la cual el usuario podra interactauar para ejecutar las funciones del sistema.

## Dashboard

![imagen del dashboard](/Practica1/doc/img/Dashboard.jpg)

En una sola pagiona se despliegan todas la funciones de la pagina web.

### 1- Ver ciudades

En la parte media a la izquierda se alcanza a ver una lista desplegada con todas las ciudades qeu estan cargadas en el sistema. 
Por dar un ejemplo tenemos a:

* Berlín
* Munich
* Colonia
...

### 2- Agregar ciudad

En esta barra de texto, ubicada en la izquierda superior, podemos agregar el nombre de una ciudad, este nombre no deberá ser igual al de una ciudad ya ingresada.

Para guardar la ciudad le damos al btón de "Agregar".

### 3- Conectar Ciudades

Ubicado en la parte superior derecha encontramos una barra de texto con un botón con la función de agregar una conexión entre dos ciudades, para esto se nos desplegaran dos listas de opciones para una ciudad o otra (no se puede elegir una misma ciudad) tambien agregamos la distancia en kilometros de la conexión entre las ciudades.

Finalmente hacemos click sobre el botón de "Enlazar" para enviar la conexión a ser guardada.

### 4- Recorridos Inteligentes entre Ciudades

En la parte de abajo a la derecha tendremos el apartado más inovador que implica la importancia del sistema.

En este espacio selecionamos dios ciudades (las cuales no necesaria mente estan conectadas directamente).  Despúes podemos elegir entre que nos muestre la mejor ruta, la más corta, para conectar estas ciudades o podemos elegir que se nos desplieguen todas las rutas con sus distancias totales.

Para el caso de la imagen se ve el de elegir todas los recorridos posibles, aqui se puede observar que se despliega todo el camino de ciudades y la distancia total.
