# Manual de Usuario

En este manual se desribira elfuncionamiento de las paginas web con la cual el usuario administrador podra interactauar para ejecutar las funciones del sistema.

## Dashboard principal

### 0- Login

![imagen login](/Practica2/documentation/login.png)

Login donde el administrador tendra que ingresar las credenciales correctas, se deben ingresar con el usuario IA1-User y la contraseña IA1 -password@_new

### 1- Categorias

![imagen login](/Practica2/documentation/categories.png)

En esta pagina se podran CREAR, EDITAR, LEER Y ELIMINAR las categorias de la base de datos. Se empieza con un despliegue y arriba de este un espacio para ingresar el nombre de una categoria a crear.
...

### 2- Preguntas

![imagen login](/Practica2/documentation/preguntas.png)

En esta pagina se podran CREAR, EDITAR, LEER Y ELIMINAR las preguntas presentes en la base de datos, eligiendo la categoria a la que pertenecen. Al igual que con las categorias se despliega la lista de pregutnas que hay y arriba se presenta un formulario para crear otra, ingresando su pregunta, respuesta y categoria.

### 3- Cambiar id del chat del bot

![imagen login](/Practica2/documentation/telegramconfig.png)

En la parte de en medio tenemos una barra donde esta el espacio para cambiar el id del bot, este funciona con un boton abajoel cual envia el cambio de id.

### 4- Extras

Para la navegación se presenta una barra lateral a la izquierda por la cual se cambia entre las paginas del sitio.

Al princio de haber ingresado el login redirige a una pagina dond ese selecciona a que sitio se desea ir entre los mencionados anteriormente.
